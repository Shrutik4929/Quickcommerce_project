# Quickcommerce_project-Analytical Project

- This project focuses on analyzing a quick commerce dataset to understand key business aspects such as delivery performance, customer behavior, and revenue trends. The goal is to transform raw data into meaningful insights and support data-driven decision-making.
The project follows an end-to-end data analysis workflow using Python, MySQL, and Power BI, covering data cleaning, querying, analysis, and visualization.

- Objectives:
  
1.Analyze delivery efficiency using delivery time and distance
2.Understand customer behavior and rating patterns
3.Evaluate company performance based on revenue and orders
4.Study the impact of discounts and payment methods
5.Build an interactive dashboard for business insights

- Tools & Technologies:
  
1.Python (Pandas) → Data cleaning and preprocessing
2.MySQL → Data storage and SQL-based analysis
3.Power BI → Dashboard creation and visualization
4.DAX → KPI and measure creation

- Outcome:
  
The project provides a comprehensive business analysis dashboard that helps identify operational inefficiencies, understand customer preferences, and support better strategic decisions.
